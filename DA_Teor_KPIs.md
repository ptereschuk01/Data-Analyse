
1. Финансовые KPI / Finanz-KPI / Financial KPI
   

| KPI            | Формула                                           | Единицы измерения | Deutsch                                                | English                                 |
| -------------- | ------------------------------------------------- | ----------------- | ------------------------------------------------------ | --------------------------------------- |
| Выручка        | ∑ продаж                                          | €                 | Umsatz                                                 | Revenue                                 |
| Прибыль        | Выручка − Расходы                                 | €                 | Gewinn                                                 | Profit                                  |
| Маржинальность | (Прибыль / Выручка) × 100%                        | %                 | Gewinnmarge                                            | Profit Margin                           |
| Себестоимость  | Σ затрат на производство                          | €                 | Herstellkosten / COGS                                  | Cost of Goods Sold (COGS)               |
| ROI            | (Прибыль / Инвестиции) × 100%                     | %                 | Kapitalrendite (ROI)                                   | Return on Investment (ROI)              |
| EBITDA         | Прибыль до вычета процентов, налогов, износа      | €                 | EBITDA                                                 | EBITDA                                  |
| ARPU           | Общий доход / Кол-во пользователей                | €                 | Durchschnittlicher Umsatz pro Nutzer (ARPU)            | Average Revenue Per User (ARPU)         |
| ARPPU          | Доход от платящих / Кол-во платящих пользователей | €                 | Durchschnittlicher Umsatz pro zahlendem Nutzer (ARPPU) | Average Revenue Per Paying User (ARPPU) |

| KPI                             | Формула                                          | Единицы измерения | Deutsch                              | English                  |
| ------------------------------- | ------------------------------------------------ | ----------------- | ------------------------------------ | ------------------------ |
| Время обработки заказа          | ∑(Время обработки всех заказов / Кол-во заказов) | часы / дни        | Bearbeitungszeit von Bestellungen    | Processing Time          |
| Производительность сотрудников  | Выполнено задач / Часы работы                    | единицы / час     | Mitarbeiterproduktivität             | Employee Productivity    |

| Процент ошибок / брака          | (Кол-во ошибок / Всего операций) × 100%          | %                 | Fehlerquote / Ausschussrate          | Error Rate / Defect Rate |
| Оборачиваемость склада          | Продажи / Средний запас                          | раз               | Lagerumschlag                        | Inventory Turnover       |



3. Клиентские KPI / Kunden-KPI / Customer KPI
   

| KPI                               | Формула                                                   | Единицы измерения | Deutsch                                     | English                  |
| --------------------------------- | --------------------------------------------------------- | ----------------- | ------------------------------------------- | ------------------------ |
| Удовлетворенность клиентов (CSAT) | (Σ оценок / Кол-во опрошенных) × 100%                     | %                 | Kundenzufriedenheit                         | Customer Satisfaction    |
| NPS                               | % промоутеров − % критиков                                | %                 | Net Promoter Score (NPS)                    | Net Promoter Score (NPS) |
| % возвратов                       | (Возвраты / Всего заказов) × 100%                         | %                 | Retourenquote                               | Return Rate              |
| Средний чек (AOV)                 | Выручка / Кол-во заказов                                  | €                 | Durchschnittlicher Bestellwert              | Average Order Value      |
| Кол-во новых клиентов             | ∑ новых клиентов за период                                | чел               | Anzahl neuer Kunden                         | New Customers            |
| CLV                               | Средний доход на клиента × Средняя длительность отношений | €                 | Kundenlebenszeitwert (CLV)                  | Customer Lifetime Value  |
| ARPU                              | Общий доход / Кол-во пользователей                        | €                 | Durchschnittlicher Umsatz pro Nutzer (ARPU) | Average Revenue Per User |
| Churn Rate                        | (Ушедшие клиенты / Всего клиентов) × 100%                 | %                 | Abwanderungsrate                            | Churn Rate               |
